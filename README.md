# React Practice

A base framework for practicing with react.

0.1 Beta 👏👏👏

## Technology stack

react@16.8.2
webpack@4.29.4

## Features

- Run a project with no edit.
- Zero package, edit free.

## Environment Support

* modern browser

## How to Start

```bash

git clone --depth 1 https://github.com/duxianwei520/react.git  

cd react-practice

npm install

npm start

```
